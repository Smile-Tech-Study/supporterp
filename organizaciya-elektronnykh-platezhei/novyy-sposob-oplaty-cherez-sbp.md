**СБП (Система быстрых платежей)** -- способ оплаты, при котором клиент оплачивает абонемент через своё банковское приложение без ввода данных карты.

В **Education ERP** клиенты могут оплачивать абонементы через СБП в личном кабинете. 

:::note 

Функция доступна только для юридических лиц, подключённых к **Т-Банку**.

:::

## Как видит оплату клиент

При выборе оплаты СБП интерфейс зависит от устройства клиента:

### Если клиент оплачивает с компьютера

-  Система отображает **QR-код**.

-  Клиент открывает мобильное банковское приложение, выбирает оплату по QR и сканирует код.

-  После успешной оплаты клиент возвращается в личный кабинет.

### Если клиент оплачивает с телефона

-  Система показывает **список доступных банков** (банковские приложения).

-  Клиент выбирает свой банк -> происходит переход в банковское приложение.

-  После подтверждения платежа клиент возвращается в личный кабинет.

:::info 

Приём платежей через СБП доступен только для **юридических лиц Т-Банка**.

Остальные способы оплаты (банковская карта) продолжают работать без изменений.

:::

:::tip 

После подключения юридического лица и онлайн-оплат через **Т-Банк**, **СБП становится доступной по умолчанию**.

:::

## **Условия доступности СБП**

СБП будет доступна клиентам, если выполнены условия:

1. У школы подключены **онлайн-оплаты через Т-Банк** (через юридическое лицо).

2. Юридическое лицо выбрано в настройках онлайн-оплат школы.

3. Оплата происходит по абонементу, доступному для покупки в личном кабинете клиента (например, по ценнику или назначенному абонементу, который можно оплатить).
